# -Monitoramento-Inteligente-de-Trafego-em-Tempo-Real
Sistema de monitoramento inteligente em tempo real usando YOLOv8, ByteTrack e Streamlit. Detecta e rastreia pessoas, veículos e animais, gera trajetórias, heatmaps, contagem automática e identifica anomalias comportamentais a partir de vídeos ou streams do YouTube.
